### ⚙️ 주요 실습 내용

#### 1. 와이어샤크(Wireshark) 패킷 분석
* **인터페이스 선택**: 로컬 영역 연결 등 본인의 네트워크 인터페이스를 통해 패킷 수집 시작
* **TCP 헤더 확인**: 포트 번호, 시퀀스 번호 등 TCP 헤더의 상세 정보 분석



#### 2. 소켓 프로그래밍 핵심 (LAB 01~03) 
* **LAB_01 (엔디안 변환)**: `htons`, `htonl` 함수를 이용해 호스트와 네트워크 바이트 순서 변환
* **LAB_02 (주소 변환)**: `inet_addr`, `inet_ntoa`를 이용해 문자열 IP와 정수형 주소 간 변환
* **LAB_03 (UDP 통신)**: 비연결형 프로토콜인 UDP의 `sendto`, `recvfrom` 함수를 사용한 데이터 송수신
